# LangGraph MonoRepo Portfolio

포트폴리오 웹사이트: https://USERNAME.github.io/page/

## 프로젝트 구조
- `docs/` - GitHub Pages 정적 사이트 파일
  - `index.html` - 메인 페이지
  - `assets/` - CSS, JS 리소스
  - `404.html` - 404 에러 페이지

## GitHub Pages 설정
1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: main / docs 폴더 선택
4. Save
