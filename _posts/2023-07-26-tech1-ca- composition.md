## 컴퓨터의 구성

![image](https://blog.kakaocdn.net/dn/dODtbE/btqFA5PgvPq/G1lucknryfv0gsdkxpMSb1/img.png)

기본적으로 컴퓨터는 하드웨어와 소프트웨어로 구성된다.

## 하드웨어

CPU , 기억 장치 , 입출력 장치

CPU -> 연산과 제어를 담당

기억장치 -> CPU에서 수행할 작업들을 적재해놓는 공간

입출력 장치 -> 사용자와의 상호작용을 위해 필요한 장치들

## 소프트웨어

시스템 소프트웨어 : 운영체제, 컴파일러
응용 소프트웨어 : 엑셀, 게임 프로그램 

## 시스템 버스

버스란 컴퓨터 내에서 물리적으로 데이터가 통로가 된다.

데이터 버스 , 주소 버스 , 제어 버스 로 나뉘어 진다.

데이터 버스 -> 중앙처리장치와 기타 장치 사이에서 데이터를 전달하는 통로

주소 버스 -> 데이터를 정확히 전달하기 위해서 기억장치 주소를 정해주는 통로

제어 버스 -> 주소 버스와 데이터 버스는 모든 장치에 공유되기 때문에 이를 제어하는 수단으로 제어 신호가 필요하다. 이 제어 신호를 나르는 통로.

제어 신호 -> 제어 신호는 기억장치 읽기 쓰기, 인터럽트 , 클락 등을 뜻한다.

## 작동방식

컴퓨터는 기본적으로 읽고 처리하고 저장하는 방식으로 작동한다.

운영체제의 bit수에 따라 RAM 으로부터 데이터를 한번에 받아오는 bit의 수가 정해짐.
