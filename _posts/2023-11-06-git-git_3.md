## 깃의 3가지 영역

작업 영역 + 인덱스(Index) + 헤드(HEAD)
이 3가지 영역을 일치시켜야 한다.(동기화)

## 작업 영역

= 프로젝트 폴더. 

## 인덱스

작업 영역 폴더 내부에 있는 파일들을 인덱스에 옮긴다(ADD)  

## 헤드

최종적으로 남기는 목적지. 여기까지 오면 동기화 완료. (commit) 영구히 보관.

브랜치구조로 저장.
